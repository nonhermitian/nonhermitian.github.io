kaleidoscope.qiskit.overload.unitary¶

unitary(self)[source]¶

Return the unitary for the circuit (if any).

Returns

Unitary for the circuit.

Return type

Operator

Example

from qiskit import QuantumCircuit
import kaleidoscope.qiskit

qc = QuantumCircuit(3)
qc.h(0)
qc.cx(0, range(1,2))

qc.unitary()
